# README

## Creating a project

create a new project in {project-name}

```shell
bunx sv create {project-name}
```

## Developing

start the server and open the app in a new browser tab

```shell
cd {project-name}
bun install
bun run dev -- --open
```

## Building

create a production version of your app

```shell
bun run build
```

preview the production build

```shell
bun run preview
```
