# svelte-setup-series

## Using Node.js

Required: `{"node":"^16.14 || >=18"}`

```bash
npm create svelte@latest <project-name>
cd <project-name>
npm install
npm run dev -- --open
```

## Using Bun

Required: `bun`

```bash
bun create svelte@latest <project-name>
cd <project-name>
bun install
bun run dev -- --open
```
